# Nhã Việt - Fine Dining Restaurant SPA 🥢

Dự án website nhà hàng cao cấp "Nhã Việt" được thiết kế và phát triển dành riêng cho một người bạn. Dự án tập trung vào trải nghiệm người dùng (UX/UI) sang trọng, mang đậm bản sắc văn hóa Việt Nam kết hợp với luồng thao tác hiện đại.

## 🌟 Tính năng nổi bật (Key Features)

* **Single Page Application (SPA):** Chuyển trang mượt mà không cần reload nhờ logic thuần JavaScript.
* **Hệ thống đa ngôn ngữ (i18n):** Chuyển đổi mượt mà giữa Tiếng Việt và Tiếng Anh không độ trễ.
* **Luồng đặt bàn (Booking Flow) phức tạp:** * Chọn tầng với hình ảnh hiển thị động.
    * Lịch tương tác (Interactive Calendar) tự code.
    * Sơ đồ chọn bàn trực quan (Seat Map).
* **Thanh toán mô phỏng:** Tích hợp cổng thanh toán PayPal Sandbox.
* **Giao diện cao cấp:** Hiệu ứng Custom Cursor, Video Background, và layout Zigzag chuẩn tạp chí.

## 🛠 Công nghệ sử dụng (Tech Stack)

* **Frontend:** HTML5, CSS3 (Flexbox/Grid, CSS Variables, Animations).
* **Logic:** Vanilla JavaScript (ES6+).
* **Integration:** PayPal JS SDK.
* **Fonts & Media:** Times New Roman, Unsplash, Pixabay.
